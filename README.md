## API Reference

#### Chapters

```http
  GET /api/v1
```

| Path | Method    | Description                |
| :-------- | :------- | :------------------------- |
| `/chapters` | `GET` | **Chapters**. Return list all chapters|
| `/chapters/{id}` | `GET` | **Chapters**. Return a chapter by id
